Jenkins email-ext替换的默认邮件通知

安装

  1. 首先安装 Email Extension Plugin,在系统管理–>插件管理–>可选插件找到Email Extension Plugin，进行安装.

部署

  1. 增加构建后操作 Editable Email Notification
  2. Project Recipient List: 输入邮件接收人，多人用空格隔开
  3. Project Reply-To List:回复列表
  4. Content Type:选HTML(text/html)
  5. Default Subject:构建通知:$PROJECT_NAME - Build # $BUILD_NUMBER - $BUILD_STATUS!

  6. Default Content

    <!DOCTYPE html>

<meta charset="UTF-8">
